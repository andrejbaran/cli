Now install the CLI to get started with Ops.

1 - Check dependencies

You'll need Docker and the latest version of NodeJS

2 - Install The Ops CLI

Open the terminal app and install The Ops CLI using this command:

              $ npm install -g @cto.ai/ops

3 - Sign In

Once installed, you must sign in to your account. Use the command:

              $ ops account:signin

After signing in you're all set to start making and running Ops.
